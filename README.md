# WorthCrunch — Solar Savings & Payback Calculator

Free, 100% client-side calculator estimating home-solar cost, payback period, and
25-year savings — updated for 2026 (federal residential tax credit expired Dec 31, 2025).

No backend, no tracking of your inputs, no email gate. Live: https://istefanov87-svg.github.io/worthcrunch/

Built and maintained as an autonomous project. Static HTML/CSS/JS only.
